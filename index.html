Eduardo Paul Contardi Soria

Tengo algunos años, me gusta programar, los videojuegos, programar videojuegos y demás.

Vivo en Buenos Aires, Argentina. Estoy estudiando Ingeniería en Informática en la Universidad de Buenos Aires. Me interesa todo lo que tenga que ver con informática, desde la producción y edición de contenido multimedia, pasando por el desarrollo de videojuegos, hasta el desarrollo web full stack.

Proyectos

Una lista de los proyectos más grandes o importantes en los que he trabajado.

Abr. 2012
Canal de YouTube
VEGAS Pro | Audacity | OBS Studio

Mis primeros pasos fueron con mi canal de YouTube. Trabajé en él por 5 años, aprendiendo edición, post producción, guionado, manejo de redes sociales y demás.

Jul. 2013
Mod de Half-Life 2
C++ | Visual Studio | Source Engine

Este fue mi primer gran proyecto de desarrollo de videojuegos, que a pesar de haber quedado inconcluso, me dio los conocimientos básicos sobre POO en C++, modelado 3D, diseño de nivel, desarrollo de inteligencia artificial y mucho más.

Dic. 2016
Aplicación administrativa de escritorio
Java | NetBeans

Una aplicación de escritorio, escrita en Java, que servía para la administración de una feria de artesanos; permitía llevar conteo de los participantes, sus rubros, asistencias y organizar fechas.

Dic. 2019
Videojuego de físicas para Android
Java | Android Studio

Luego de haber desarrollado el mismo concepto pero con HTML y JavaScript, desarrollé, para una persona muy especial, un juego de físicas para Android. Éste me permitió conocer el terreno del desarrollo de aplicaciones móviles, además de permitirme explorar Android Studio.

May. 2020
Sitio web con Laravel
PHP | jQuery | Bootstrap | Laravel

Visitar sitio

Este sitio me fue encargado por una empresa de servicios técnicos, los requisitos eran poder tener un sitio web oficial donde publicitar dichos servicios, además de un sistema de turnos para la atención presencial de clientes. Personalmente decidí agregar un sistema de usuarios exclusivo para la administración del sitio. Con este proyecto pude aprender en profundidad el funcionamiento de Laravel, la arquitectura MVC y el desarrollo de sistemas CRUD.
