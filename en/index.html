Eduardo Paul Contardi Soria

I am a some years old guy who likes programming, video games, programming video games and more.

I live in Buenos Aires, Argentina. I'm studying Information Systems Engineering at Universidad de Buenos Aires. I like everything related to software and computers, from production and editing of multimedia, through video game development, to full stack web development.

My main goal is to bring software solutions to every possible aspect of life making use of the perfectionism I have; automate tasks, ease tedious actions, create useful and entertaining experiences, and, of couse, keep learning.

Projects

Here is a list of the most important or biggest projects I have worked on.

Apr 2012
YouTube Channel
VEGAS Pro | Audacity | OBS Studio

From an early age I got interested in modern multimedia content on the Internet. I worked on my YouTube channel for around 5 years. With this project I experimented with audio and video editing, quickly working, learning professional sound and image processing techniques, and to manage social networks to reach out more people.

Jul 2013
Half-Life 2 Mod
C++ | Visual Studio | Source Engine

I had try smaller projects with other games in the past, so I ventured into developing a Half-Life 2 mod with Source Engine. The mod's story was set in the same universe of the original game, Half-Life 2, but told from the point of view of one of the enemies. This project allowed me to develop my skills in 3D modeling, level design, object-oriented programming in C++, artificial intelligence design and development, story writing, and much more. Due to studies I was progressively forced to abandon the development (although I never officially canceled it...); the project was active for approximately 4 and a half years.

Dec 2016
Desktop Admin App
Java | NetBeans

I developed a Java desktop app for a the administration of a craft fair. This app allows you to keep track of how many participants were there on each date and group them by craft item. This project allowed me to learn object-oriented programming in Java, desktop apps development, and GUI design for desktop apps.

Dec 2019
Physics Game for Android
Java | Android Studio

After creating the same concept with HTML and JavaScript, I developed, specifically for a very special person, a physics game for Android that consists of bouncing a ball that looks like a pig against the edges of the screen. This project taught me all the basics of Android app development.

May 2020
Laravel Website
PHP | jQuery | Bootstrap | Laravel

Visit website

This site had to contain a brief description of the owner company, contact info and a system of turns for face-to-face client service. Thanks to this I was able to test the potential of Laravel; I developed an administrator mode that allows you to register, cancel and modify turns, and changes are updated on screen in real time. Also, clients can check their turns with an ID and cancel them if they want to. For this I used websocket technologies, complex AJAX requests, database table dependencies, security protocols to protect sensitive data and much more.
